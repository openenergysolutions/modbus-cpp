modbus::UnitIdentifier class

Modbus unit identifier.

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

Represents the 8-bit unit identifier used in the MBAP messages. When sending a request, this identifier must fit with the one reported by the slave. This feature is mainly used in Modbus+ and Modbus serial lines. However, it must be set properly on TCP/IP only sessions too.
