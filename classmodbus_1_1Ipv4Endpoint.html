modbus::Ipv4Endpoint::Ipv4Endpoint(const std::string& hostname, uint32_t port)

Constructor.

Parameters
hostname Hostname. The hostname can be an IP address or a resolvable name.
port Port number. Modbus devices should normally communicate via port 502.

Example:

// IP address endpoint
auto Ipv4Endpoint ip_endpoint{"127.0.0.1", 502};

// DNS endpoint
auto Ipv4Endpoint dns_endpoint{"foo.com", 8888};

bool modbus::Ipv4Endpoint::operator==(const Ipv4Endpoint& other) const

Checks if two endpoints are the same.

Parameters
other Other endpoint to compare
Returns true if the endpoints are the same, false otherwise.

For two endpoints to be equal, they must have the same hostname string and the same port.

Note

An endpoint with the IP address of a resolved DNS endpoint are not considered equal. This method only does a string comparison.

Checks if two endpoints are different.

Parameters
other Other endpoint to compare
Returns true if the endpoints are different, false otherwise.

For two endpoints to be different, they must have a different hostname string and/or a different port.

Note

An endpoint with the IP address of a resolved DNS endpoint are considered different. This method only does a string comparison.
