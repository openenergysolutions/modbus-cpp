modbus::LoggerFactory class

Factory of loggers.

Contents

  • Reference
    • Public static functions

Creates loggers using spdlog. Note that the spdlog is completely private dependency and you don't need it, except if you compile from the sources.

If you are already using spdlog in your application, you can use create_custom_logger(std::shared_ptr<spdlog::logger> custom_logger) to use an existing logger.

The console logger will be colored and is completely thread-safe. It uses spdlog::stdout_color_mt under the hood.

static std::shared_ptr<Logger> modbus::LoggerFactory::create_null_logger(const std::string& name)

Creates a logger that simply drops all the messages.

Parameters
name name associated with the logger
Returns Shared pointer of the logger

This logger is useful if you don't want to provide logging (e.g. in tests). It simply ignores all the messages passed to it. It uses a spdlog::sinks::null_sink_st under the hood.
