template<typename T>
modbus::SingleTypeDatabase class

In-memory database containing a single type of data.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

This class is used by InMemoryDatabase but it can be particularly useful when implementing a custom IServerSession. It provides methods to add, get and set values individually or in sequence. It also provides methods to validate ranges of addresses.

See also

BitDatabase, RegisterDatabase
